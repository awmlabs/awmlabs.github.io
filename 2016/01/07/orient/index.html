Exif Orientation の操作方法

スマートフォン等のカメラで撮影した時に、その向きを記録するメタデータです。

JPEG には画像データそのものと別に Exif という形式でメタデータ(日付とか撮影条件とか場所とか、写真をとった時の向きとか)を持つ事ができます。

Orientation はこのうち、写真をとった時の向きを指します。

(サンプル。iPhone の中に何か映ってる図、左右上下がわかるようなもの)

詳しくはこちら > http://hackmylife.net/archives/7400448.html

ImageMagick を使いたい所ですが ExifTool が便利なのでこっちを使う事にします。

ExifTool

MacOS なら

$ brew install exiftool

Debian Linux なら

apt-get install exiftool

でインストールできます。
