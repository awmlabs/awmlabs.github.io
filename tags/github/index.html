Blogはじめました

published on January 01, 2016
Blog はじめました WordPressを試したり色々と迷いましたが、神のお告げに従い Github.io + Hugo + CircleCI の構成にしました。 https://twitter.com/nobu666/status/680683739534393344 Github.io github.com で <ユーザ名>.github.io というレポジトリを作れば、その URL でアクセス出来ます。 https://github.com/awmlabs/awmlabs.github.io => https://awmlabs.github.io/ CNAME をつけて awm.jp からアクセスできるようにしました。 => https://blog.awm.jp/ (awmlabs.github.io を指します) ここにブログのHTMLを入れていけば良い訳です。 md ファイルからBlogサイトのHTMLを生成するツール Hugo を使ってみました。 Hugo $ hugo new site blog.awm.jp $ cd blog.awm.jp $ hugo new 2016/01/01/blog.md のようにすると blog.awm.jp/content/2016/01/01/blog.md にテンプレートの md ファイルが出来るので、そこに文章を記述して、blog.awm.jp フォルダのトップで、 $ hugo を実行します。すると public 以下に表示用の HTMLファイルが生成されます。 実際に表示する為には、テーマが必要です。 Hugo themes http://themes.gohugo.io/ ここで色々なテーマが確認できます。base16 に一目惚れしたので、これを適用しました。 http://themes.gohugo.io/base16/ 弄るファイルがあるのでまず自分のアカウント(awmcorp) に fork して。 git Read More...
